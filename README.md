# LingoAI Translator

A fully functional, modern, responsive AI-powered language translation website.

## Features
- **Supported Languages:** Translate between English and 20+ other languages (Hindi, Kannada, Tamil, Telugu, Malayalam, Marathi, Bengali, etc.)
- **Text & Voice:** Built-in Speech-to-Text (Microphone) and Text-to-Speech capabilities.
- **Modern UI:** Built with React, Tailwind CSS v4, and Framer Motion for smooth animations and a beautiful glassmorphism design.
- **Privacy First:** Translation history is saved locally in your browser.
- **Developer Ready:** Uses Vite for lightning-fast build times.

## How to run locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to `http://localhost:5173`

## Tech Stack
- React 19
- Vite
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React (icons)
- Google Translate Unofficial API
